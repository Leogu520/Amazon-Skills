# 关键词与属性优先级

关键词用于提高准确发现能力，而不是在标题中堆入最多索引短语。

## 证据来源

按以下顺序优先使用：

1. 结构化商品事实和当前 Product Type Definition
2. ASIN 级 Search Query Performance
3. 包含订单、销售额、转化和花费的广告搜索词报告
4. Top Search Terms 和 Product Opportunity Explorer
5. 现有自然搜索词和 Listing 表现历史
6. 竞品措辞和自动补全，仅作为发现线索

不得把竞品声明当作用户商品的事实。

## 优先级分类

### P0：必要身份和防误购信息

包括核心产品类型、型号、数量、尺寸、容量、接口、适配关系、年龄或体重范围和变体身份。

买家需要在点击前确认的信息应保留在标题中。P0 的优先级高于搜索量。

### P1：核心搜索词

能够准确命名产品的最高意图搜索词。实际客户常用表达与产品类型一致时，优先采用客户措辞。

### P2：主要差异点

一个会改变比较行为且已经验证的功能或材料，例如 `65W GaN`、`Leakproof` 或 `Stainless Steel`。

### P3：次级发现词

使用场景、次要材料、次要功能和相关同义词。空间不足时迁移到 Item Highlights、五点、描述或后台词。

### P4：删除

促销、主观形容词、无关高流量词、重复同义词、无证据声明，以及亚马逊已能处理的拼写或单复数变体。

## 数据解读

- Search Query Performance 可向符合条件的品牌卖家展示搜索量、曝光、点击、加购和购买。
- Sponsored Products 搜索词报告只包含至少发生一次广告点击的词，可用于验证付费搜索表现，但不能代表完整自然需求。
- 高曝光低点击可能意味着词过宽，或搜索结果首屏竞争力不足。
- 高点击低转化可能意味着流量错配、限定信息缺失、价格问题或详情页问题。
- 精确识别 ASIN 且转化强的低流量词，也可能值得占用标题空间。

## 选择测试

对每个拟写入标题的词，依次判断：

1. 是否准确描述这个 ASIN？
2. 是否帮助识别商品或防止买错？
3. 是否有搜索贡献证据？
4. 能否用更少的标准字符表达同一含义？
5. 是否更适合放在其他 Listing 字段？

删除历史重要关键词时，记录判断理由。
